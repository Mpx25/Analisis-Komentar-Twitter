# Analisis-Komentar-Twitter

Repository ini berisi project pengumpulan dan analisis data dari Google Play Store dan Twitter. Tujuan proyek ini adalah untuk memahami sentimen pengguna, tren aplikasi, dan pola interaksi sosial media melalui data ulasan dan komentar yang dikumpulkan.

# Fitur Utama:

Data Collection: Mengambil data aplikasi dan review dari Google Play, serta komentar Twitter menggunakan API dan teknik scraping.

Data Preprocessing: Membersihkan dan menormalisasi data untuk analisis yang akurat.

Analisis Data & Sentimen: Mengidentifikasi opini pengguna, tren aplikasi, dan insight dari komentar.

Visualisasi: Menyajikan hasil analisis dalam bentuk grafik dan dashboard interaktif untuk memudahkan interpretasi data.

# Tujuan:
Memberikan insight yang dapat membantu pengembang aplikasi memahami opini pengguna dan mendukung pengambilan keputusan berbasis data.
